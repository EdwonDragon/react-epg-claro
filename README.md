## Technologies
***
* [VITE](https://vitejs.dev/): Version 3 
* [React](https://es.reactjs.org/): Version 18.2.0
* [React Bootstrap](https://react-bootstrap.github.io/): Version 2.7.0
* [Planby](https://planby.netlify.app/)
